WEBSITE

TEMA BEKSRIVELSE

Grundlæggende web omhandlede teorier, principper og basale frontend-værktøjer. I forløbet blev vi præsenteret for figma, samt kodesprogene HTML og CSS, som alle skulle bruges til at udvikle vores første responsive-hjemmeside.

PROJEKT BESKRIVELSE

Studieretningsprøven, var en opgave der skulle teste vores evner inden for udvikling af en responsiv hjemmeside. Ved brug af wireframe, skulle man udarbejde en hjemmeside beregnet til henholdsvis mobil og PC.

PROCESS

HTML

Ved at arbejde med diverse HTML øvelser, får man en forståelse af strukturen, i form af opstilling af header, body og footer. Vi lærer også omkring terminologi og valg af skrifttyper.

CSS

CSS er måden vi styler vores HTML på og vi får i undervisning information om pseudo-klasser og andre metoder som kan bruges til at tilrettelægge vores website. Vi lærer også om media-queries som giver muligheden for at opnå en responsiv website.

GRID

Der bliver arbejdet med GRIDS for at opfylde opgavekravene og hermed kunne give websiten det rigtige layout.

STUMBLEBLOCKS

CSS og grids krævede meget forståelse og koncentration. Det var en hård process at få det lært.
